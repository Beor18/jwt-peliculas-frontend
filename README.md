## Jwt Elixir - Frontend realizado con React

### Características:

- Redux
- Json web Tokens
- Login de usuarios
- Registro usuarios
- Perfil de usuario
- Rutas protegidas
- Mostrar lista de datos

Utilizando el backend [`Jwt, autenticación y crud API Rest con Elixir 1.7.3 y Phoenix 1.4`](https://github.com/Beor18/jwt-elixir)
